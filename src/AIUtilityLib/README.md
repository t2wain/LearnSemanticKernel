# Introduction

The purpose of this library is to explore systematically the various data structures and methods of the Semantic-Kernel (SK) API.

The various utility classes (**xxxUtility.cs**) are designed to document the main SK methods that are most useful in an application:
- ChatMessageUtility.cs
- KernelFunctionUtility.cs
- KernelUtility.cs
- PromptUtility.cs

These classes implement many **Exlorexxx** methods to drill down and examine the various data structures implemented in the SK libraries

Other classes document the basic pattern of an AI chat session:
- ChatService.cs
- ChatSession.cs
- KernelFactory.cs

The various README files document the essential assemblies, namespaces, classes, and methods of the SK libraries:
- README_Sk.md
- README_AltExt.md
- README_Google.md
