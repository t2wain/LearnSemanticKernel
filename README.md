# Learning GenAI Development with dotNET

The objective of this project is to learn how to develop typical features of a Generative AI application. These features have been discussed widely in many publications. Such features include:

- Prompt Engineering
- Prompt Templating
- RAG
- Tool Calling
- Agentic AI
- Agentic Workflow
- Model Context Protocol (MCP)

# LangChain and LangGraph

I started my AI development learning with the Python libraries LangChain and LangGraph. There are abundant documentation and examples on the use of these libraries. I feel this effort establishes the foundation in my AI development learning.

# Semantic Kernel

I am a dotNET developer and so I prefer to do development in C# and Visual Studio IDE. I began my AI development in dotNET with the use of the Microsoft Semantic Kernel library. A sense of familiarity about the Microsoft library APIs since it is somewhat similar to the Python libraries.

# Microsoft Agent Framework

As I began to learn about the development of Agentic workflow, I discovered that there is another dotNET library that will supersede the Semantic Kernel library. This libary is the Microsoft Agent Framework library.
